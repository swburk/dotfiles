# Dotfiles

## Requirements

- Ag (The Silver Searcher)
- Git
- Ranger
- Tmux
- Vim
- Zsh

## Install

Download and run the install script:

    bash -c "`curl -fsSL https://raw.github.com/samuelburk/dotfiles/master/bootstrap.sh`"

## Update

Run the install script again to update:

    ~/.dotfiles/install.sh
